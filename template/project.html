Face Recognition Attendance System

This project is a smart attendance system built using Python, Flask, OpenCV, and SQLite3. It enables real-time face recognition for marking employee or student attendance automatically. The system includes separate modules for registration (with photo capture), daily attendance via face or ID, and monthly report generation with filters for morning and evening sessions.

The project also supports exporting attendance reports and features a user-friendly web dashboard for administrators.

U-fill Academy

This is a team-based ongoing project aimed at developing a fully responsive and interactive website for U Fill Academy, an educational institution offering AI and technical training programs. The project focuses on creating a clean, modern UI with smooth navigation and dynamic content presentation to improve the student enrollment experience. It includes modern sections like Home, About, Courses, Gallery, Achievements, and Contact.

As part of the frontend development team, I am responsible for building responsive page layouts, implementing animations, and integrating features like dark/light mode and interactive forms. The project is currently under active development and will include both frontend and backend integration upon completion.
